# Jarvis AI Voice Assistant 🤖

AI-powered Voice Assistant built using Python, SpeechRecognition, and OpenAI API.

## 🚀 Features
- Voice recognition
- AI response generation
- News headlines integration
- Music control
- Web automation

## 🛠 Tech Stack
- Python
- SpeechRecognition
- OpenAI API
- NewsAPI

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your API keys
3. Run:
   python main.py
